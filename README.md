# china-wccup26-guide

本仓库用于归档和发布多个独立的 HTML 页面。所有页面均为静态文件，不依赖于特定域名或外部服务，仅供查阅和参考。

## 项目简介

该项目收集并整理了一系列独立 HTML 页面，这些页面内容涉及不同领域的说明、指南或信息展示。仓库本身不指向任何具体网站或服务，仅作为 HTML 文件的集中存放和版本管理用途。

## 目录说明

```
.
├── README.md           # 本说明文件
├── pages/              # 存放所有 HTML 页面的主目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 页面依赖的静态资源（如 CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/`：所有独立 HTML 页面均放置于此，文件名按顺序或主题命名。
- `assets/`：存放页面共用的样式、脚本、图片等资源，按类型分目录管理。
